Before you start

To pay using a one-off Direct Debit payment you’ll need to have the following bank details ready:

  • Account number
  • Sort code
  • Name on account

If more than one person is required to authorise Direct Debits on the bank account you can't proceed with this online payment method. You should use a different payment method.
